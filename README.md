## 🕶️ Conventions

**Issue** | `[플랫폼] 문제이름`  
e.g. [Softeer] 문자열 출력하기

**Branch** | `이름/YYMMDD`  
e.g. 수정/250203

**PR** | `이름 YYMMDD`  
e.g. [수정] 250203

**Commit Message** | `내용 문제이름`  
e.g. Create 문자열 출력하기  
e.g. Refactor 문자열 출력하기

**Directory/File Name** | `platform/YYMMDD_문제_이름/id.언어`  
e.g. softeer/250203\_문자열\_출력하기/kwaksj329.js
